# 📖 Sobre o Projeto

O **Sistema de Gestão Escolar** foi desenvolvido com o objetivo de modelar e implementar um banco de dados capaz de gerenciar informações de uma instituição de ensino.

O projeto contempla desde a modelagem conceitual até a implementação física do banco de dados, permitindo o gerenciamento de alunos, professores, disciplinas, turmas, matrículas e notas.

Além disso, foram aplicados conceitos fundamentais de modelagem relacional, normalização e integridade referencial.
